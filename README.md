# CraftMCP

CraftMCP is a lightweight open-source Windows design tool for human and AI collaboration.

It is intended to let AI agents create and edit simple graphic designs locally, while human users can directly refine those outputs and export them as both images and structured JSON.

## MVP Direction

- Windows-first desktop app
- Agent-driven design generation through structured commands
- Human-editable canvas, layers, and properties
- Use cases: social graphics, UI mockups, and slide-style layouts
- Local-first file handling
- PNG export for output
- JSON export for exact machine-readable design context

## Product Principle

CraftMCP is not trying to be a full design suite. The goal is to be a practical open-source tool people can use with their own AI models.
